# 🍽️ Информационная система учёта заказов ресторана

## Описание проекта

Веб-приложение для автоматизации работы ресторана.  
Система помогает управлять заказами, столами, меню, сотрудниками и сменами.

## Функциональные возможности

| Раздел | Возможности |
|--------|-------------|
| **Заказы** | Создание заказа, отмена заказа, просмотр списка |
| **Столы** | Отслеживание статуса (свободен/занят), 5 столов |
| **Меню** | Добавление блюд, удаление блюд, просмотр списка |
| **Сотрудники** | Добавление сотрудников, удаление, просмотр |
| **Смены** | Начало смены, завершение смены, просмотр |
| **Выручка** | Автоматический подсчёт за сегодня |

## Технологии

- **ASP.NET Core** — веб-фреймворк
- **C#** — язык программирования
- **SQLite** — база данных
- **HTML/CSS** — интерфейс
- **Git & GitHub** — контроль версий
